# ceasar-cipher-with-tkinter

A desktop application for encrypting and decrypting text using the Caesar cipher. Built with Python and Tkinter.

## Features
- Encrypt and decrypt text
- Keyboard shortcuts: Ctrl+E (encrypt), Ctrl+D (decrypt)
- Copy results to clipboard
- Simple passcode (shift) based encryption

## Requirements

Python 3.8+

## Installation

```
pip install -r requirements.txt
```

## Usage

Enter text and a numeric passcode (shift value). Click Encrypt or Decrypt.

Run the application with:

```
python main.py
```
